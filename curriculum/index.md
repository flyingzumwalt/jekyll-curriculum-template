## Curriculum Structure

A Curriculum is made up of Courses, Learning Modules, and Activities

```
|   |-- course-name
|   |   |-- index.md

|   |   |   |-- learning-module-name

|   |   |   |   |-- activities
|   |   |   |   |   |-- activity-name
|   |   |   |   |   |   |-- index.md
```
